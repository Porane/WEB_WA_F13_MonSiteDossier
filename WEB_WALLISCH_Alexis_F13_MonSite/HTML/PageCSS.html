Définition :

Les feuilles de style en cascade, généralement appelées CSS de l'anglais Cascading Style Sheets, forment un langage informatique qui décrit la présentation des documents HTML et XML. Les standards définissant CSS sont publiés par le World Wide Web Consortium (W3C). Introduit au milieu des années 1990, CSS devient couramment utilisé dans la conception de sites web et bien pris en charge par les navigateurs web dans les années 2000.


Les différentes propriétés de mise en forme :

Il existe plusieurs propriétés en CSS permettant de faire une multitude de mises en forme ;

  • text-align permet d'aligner le texte selon un critère comme gauche ou centre ;
  • font-family permet de mettre une police d'écriture ;
  • font-size permet de modifier la taille de la police ;
  • color permet de changer la couleur ;
  • background-color permet de changer la couleur de fond ;
  • background-image permet de mettre une image de fond ;
  • border-radius permet d'avoir des bords arrondis à un background ;
  • margin-left et margin-right permettent de mettre de la marge à gauche et à droite ;
